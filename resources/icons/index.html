    Choose the icon you need from this page, then follow the implementation instructions on the icon component page. If you are building a Lightning Component, you may require an additional Lightning helper component to use SVGs.

    action Icons

    Actions can be seen throughout the interface and represent actions a user can take on any given screen.

    custom Icons

    Custom icons are available for the identity of user created objects.

    doctype Icons

    Doctype icons represent a type of file when a preview or image is unavailable

    standard Icons

    Standard icons represent entities and objects within Salesforce.

    utility Icons

    Utility icons are used throughout the interface and are SVG’s for extensibility.
