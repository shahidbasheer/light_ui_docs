Tiles

  • With Badge
  • List
  • Board

  • Task
  • Author
  • Component Overview

A tile is a grouping of related information associated with a record.

Tiles can have different groupings of information based on its context. Pay close attention to the markup, as each tile layout is constructed differently.

To initialize the base tile, apply the .slds-tile class to a containing <div>that contains a tile title (.slds-tile__title). Include other optional elements, such as images, icons, action overflows, and description lists, within the containing <div>.

For available classes and options, see the tile overview legend below.

Basedev ready

The base tile is borderless by default.

.slds-tile {

  + .slds-tile {
    margin-top: $spacing-x-small;
  }

  &__title,
  &__detail {
    position: relative;
  }

  &__meta {
    color: $color-text-default;
  }
}

Component Overview

This table gives you a quick overview of the SLDS CSS classes that can be applied to this component.
SLDS classOutcomeRequiredApplied toComments
.slds-tileInitializes tileRequired<div>
.slds-tile__titleRequired<p>
.slds-tile__detailRequired<div> or <ul>
.slds-tile__metaApplies text color change.slds-list--horizontalOnly used on author tile
.slds-mediaLayout helper that aligns image and text.slds-tileUse on tiles with Icons, Actions and Badges
.slds-media__bodyLayout helper<div>Wraps .slds-tile__title and .slds-tile__detail inside a .slds-media object.
.slds-truncateApplies overflow truncation to tile detail itemsCan be applied to anything that needs truncation
.slds-list--horizontalHorizontally aligns list items<ul>
.slds-has-dividersAdds dotted dividers in a horizontal list.slds-list--horizontal
.slds-list--verticalVertically aligns list items<ul>Applied to the list that holds multiple tiles
.slds-gridInitializes a grid layoutWrap .slds-tile__title and icon/badge element
.slds-grid--align-spreadSpreads the elements on each end of the same axis.slds-grid
.slds-has-flexi-truncateLayout helper applied to a parent .slds-grid when .slds-truncate is within a flex element.slds-col
.slds-hint-parentHighlights dimmed action overflow icons on tile hover.slds-tile
