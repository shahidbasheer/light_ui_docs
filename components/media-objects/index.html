Media Objects

Variants

  • Base
  • Center
  • Reverse
  • Double
  • Responsive
  • Component Overview

When you need text and a figure next to each other, use a media object.

Placing text next to an image, icon, figure, or video is common for many applications. The figure can be either on the right, left, or both. The text can start at the top of the image or be centered next to it. The width of the media object is controlled either by the parent container or a secondary module class.

Basedev ready

This is the base version of the media object. The container for this object can be any HTML element. It should have the class .slds-media placed on the container. Inside are the two required elements. The figure (image, icon, etc) should be placed inside an element with the .slds-media__figure class. The information associated with it should be placed in an element with the .slds-media__body class.

By default, the text starts at the top of the figure. Use the centered variant to center the text and figure.

Centerdev ready

With the centered media object, the text remains vertically centered next to the figure. In addition to the .slds-media class, add the .slds-media--center class to the container.

Reversedev ready

To place the text to the left of the figure, use the .slds-media--reverse class. You can add the .slds-media--center class to align them in the middle.

Doubleprototype

Use the double media object to place a figure on both the right and left of the text.

As in the base media object, place both the .slds-media__figure elements before the .slds-media__body. The figure that displays on the right also receives the .slds-media__figure--reverse class.

Component Overview

This table gives you a quick overview of the SLDS CSS classes that can be applied to create media objects.
SLDS classOutcomeRequiredApplied toComments
.slds-mediaDefines the overall media objectRequiredCan be applied to any outer element and will be as wide as its parent container
.slds-media__figureDefines the figure areaRequiredApply this class to the first element inside the .slds-media and contains the image, icon, svg or video
.slds-media__bodyDefines the body areaRequiredApply this class to the second element inside the .slds-media and contains the text or other content
.slds-media--centerAligns the content in the .slds-media__body to the middle of the .slds-media__figure.slds-media
.slds-media--reverseReverses the media object so that the figure is on the right.slds-media
.slds-media--doubleAllows you to add a figure on both the right and left sides.slds-mediaBoth figures are added prior to the .slds-media__body
.slds-media--responsive.slds-media__figure and .slds-media__body stack on smaller screens.slds-media
